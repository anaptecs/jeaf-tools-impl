# JEAF Tools Implementation README #

This repository contains the implementation of JEAF Tools API. It contains default implementations for all services as well as separate projects that provide custom implementations for some parts.
 

## Links ##
For further information please refer to:

* [JEAF Tools Getting started](https://anaptecs.atlassian.net/l/c/xQwBqiwH)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.tools/jeaf-tools-impl)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.tools/jeaf-tools-impl)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
